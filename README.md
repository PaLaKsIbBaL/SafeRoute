 How to Run the Project

1. Clone the repository or download the ZIP file
2. Open the project folder
3. Open `index.html` in a browser (recommended: Chrome)
4. Enter/select source and destination
5. View the route and output on the map

 Note: An active internet connection is required for map and API loading.
